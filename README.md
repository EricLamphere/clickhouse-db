# Clickhouse Database

## Getting Started

### Install
[Installation instructions](https://clickhouse.com/docs/en/getting-started/quick-start)
```sh
curl https://clickhouse.com/ | sh
```

### Start the server!
Run **clickhouse-local**
```sh
./clickhouse
```

Start the **server**
```sh
./clickhouse server
```

Start the **client**
```sh
./clickhouse client
```
